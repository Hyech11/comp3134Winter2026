What is the definition of server port?
A server port is a logical communication endpoint used by a server application to send and receive network traffic. It helps direct data to the correct service running on a host.

What is server port range?
The standard server port range is from 0 to 65535.

How many bits does number above represent?
The port number range 0 to 65535 represents 16 bits.
